Welcome to CUNQA’s documentation!¶

CUNQA is a Python/C++ platform to simulate distributed quantum computing on the CESGA HPC environment. CUNQA creates and manages virtual QPUs, which are classical resources allocated for emulation tasks, where distributed quantum programs can be executed. The Python API is structured in a way which will be familiar to Qiskit users, while the backend is written in C++ for performance. All code available on our GitHub repository.
